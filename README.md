# Basic webpack test project

## Usage

### Installation
```
git clone https://github.com/geocolumbus/webpack-basic-notes.git
npm install
```

### Build
```
npm run build
```

### Run
```
npm start
```

### Test with Karma and Jasmine in Chrome

```
npm test
```

Open localhost:5000 in your browser.

## Features

* webpack
* uglify.js
* serve
* babel
* karma / jasmine testing
